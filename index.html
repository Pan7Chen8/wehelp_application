標題：使用 HTML + CSS 建立 GitHub Page 步驟：
1. 建立 GitHub 倉庫 (repository)
  • 登錄或註冊 GitHub 帳戶（https://github.com）。
  • 在儀表板中，點擊 "New" 建立一個新的倉庫。
  • 輸入倉庫名稱，並選擇公共或私有倉庫的選項。
  • 勾選 "Initialize this repository with a README"，以便建立一個 README.md 文件。
  • 點擊 "Create repository" 完成建立。
2. 複製倉庫到本地
  • 打開終端或命令行介面。
  • 輸入：git clone https://github.com/your-username/your-repository.git 複製剛剛建立的倉庫到本地。
3. 建立 HTML 檔
  • 在複製到本地的倉庫資料夾中，建立一個新的 HTML檔案。ex：index.html
  • 使用IDE打開html檔案，並做任何你想要的編輯
4. 建立 CSS 檔
  • 在複製到本地的倉庫資料夾中，建立一個新的 CSS 檔案。ex：style.css
  • 使用IDE打開css檔案，並做任何你想要的編輯
5. 完成編輯(只有一個分支的情況下，在ide執行以下)
  • git add：此時內容被加到暫存區，把檔案交給git，讓git開始追蹤目錄.
  • git commit -m "說明"：為此次編輯寫下說明
  • git push origin main：將此分支推到遠端名為main的分支
